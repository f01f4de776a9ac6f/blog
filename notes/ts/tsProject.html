# vite的五个环境变量

BASE_URL  //开发或生产环境服务的公共基础服务。
DEV       //当前环境是否为开发环境，true为开发环境。
MODE      //应用的运行模式。分为开发环境模式（development）、生产环境模式（production）。
PROD      //当前环境是否为生产环境，true为生产环境。
SSR       //是否为服务端渲染 true为服务端渲染。
